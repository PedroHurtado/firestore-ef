# Project Guidelines

## Git Commits
- NO incluir "Generated with Claude Code" en los mensajes de commit
- NO incluir "Co-Authored-By" en los mensajes de commit
- Commits limpios, sin atribución a Claude

## Flujo de trabajo
- Al inicio de cada sesión, leer el documento de trabajo para obtener contexto
- Después de cada commit, actualizar el documento de trabajo con el ID del commit automáticamente, sin pedir confirmación
